SYSTEM REQUIREMENTS
------------------------------------------------------------------------
- A C++ compiler (g++, clang++, or similar)
- Standard C++ libraries (iostream, vector, cmath)
- No external third-party libraries are required.

------------------------------------------------------------------------
HOW TO COMPILE
------------------------------------------------------------------------
1. Open your terminal or command prompt.
2. Navigate to the directory containing "main.cpp".
3. Run the following command to compile the code:

   g++ main.cpp -o assignment1

------------------------------------------------------------------------
HOW TO RUN
------------------------------------------------------------------------
Once compiled, run the executable using the following command:

   ./assignment1

------------------------------------------------------------------------
PROGRAM DESCRIPTION
------------------------------------------------------------------------
The program executes the solutions for the assignment in sequential order:

- Q1: Translates a MATLAB switch statement to C++.
- Q2: Implements a vector printing function (used in Q4 and Q5 output).
- Q3: Generates Fibonacci numbers < 4,000,000.
- Q4: Tests functions for prime checking, factorization, and prime factorization.
- Q5: Prints the first 8 rows of Pascal's Triangle.

All results will be printed directly to the standard output (console).
